## 介绍

学习 React 时做的个人网站，展示个人推荐电影及影评文章，独立编码/设计。

## 现状展示

<img src="./public/readme.png" width="70%" />

## 用到的库

-   React (Router/Redux,用于项目构建)
-   Lottie (用于页面动画)
-   Swiper (用于轮播图)
-   阿里图标矢量库 (用于图标资源)
-   marker (用于 markdown 展示)
-   ...

## 下载本地使用

1. npm install
2. npm run start
